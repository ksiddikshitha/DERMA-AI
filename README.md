# DERMA-AI 🏥

A comprehensive web application for skin analysis, personalized recommendations, and doctor consultations.

## Features ✨

- **🔍 Skin Image Analysis**: Upload skin concern images and get instant analysis
- **💊 Product Recommendations**: Get personalized product suggestions based on skin type
- **💬 Online Doctor Consultation**: Chat with licensed dermatologists in real-time
- **🏥 Nearby Clinics**: Locate dermatology clinics and services near you
- **📅 Offline Booking**: Schedule appointments with dermatologists

## Tech Stack 🛠️

- **Frontend Framework**: React 19.2.6
- **Build Tool**: Vite 8.0.12
- **Styling**: CSS3 with modern features
- **Language**: JavaScript (TypeScript ready)
- **Package Manager**: npm

## Getting Started 🚀

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ksiddikshitha/DERMA-AI.git
cd DERMA-AI
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Available Scripts

- **`npm run dev`** - Start the development server with HMR
- **`npm run build`** - Build the project for production
- **`npm run preview`** - Preview the production build locally
- **`npm run lint`** - Run ESLint to check code quality

## Build for Production 📦

```bash
npm run build
```

This will create an optimized build in the `dist/` directory.
